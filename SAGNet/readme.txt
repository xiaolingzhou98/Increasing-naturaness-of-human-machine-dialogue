Requirement:

Python=3.6, Tensorflow=1.3.1, pyltp=0.2.1 and numpy=1.16.2

Train and Test

Train the model,you need to download the pretrained model ltp_model.

For Mdata:
run sag_choose_train.py
run sag_choose_test.py
For Tdata:
run judge_train_.py
run judge_test_.py

if you want to calculate bleu and judge indicators:
For Mdata:
run bleu_style.py
run rouge.py
For Tdata:
run bleu_style_judge.py
run rouge_judge.py
